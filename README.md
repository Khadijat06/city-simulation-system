# city-simulation-system (C++)
## 📌 Overview
A C++ simulation that models interactions between different city regions such as residential, commercial, and industrial zones.

## 🚀 Features
- Simulates population growth and resource distribution
- Models interactions between multiple region types
- Uses file-driven configuration for dynamic input
- Iterative simulation over time

## 🛠️ Tech Stack
- C++
- File I/O
- Object-Oriented Programming

## 💡 Key Concepts
- System modeling
- Modular design
- Data-driven simulation
